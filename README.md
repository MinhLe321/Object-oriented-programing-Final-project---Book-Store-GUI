# Object-oriented-programing-Final-project---Book-Store-GUI
A GUI that can manage many books like adding, searching and deleting books :D
To run this program, first you need to install javafx

Go to this website: https://gluonhq.com/products/javafx/
Go down and dowload the sdk version that match your computer spec: I use macos so I choose macos with the type of SDK

Then move the unzip the file and move it close to the GUI.java file

then run two commands in order and it will work:
//Command that link library together: javac --module-path "javafx-sdk-26"/lib --add-modules javafx.controls GUI.java
//Command run: java --module-path "javafx-sdk-26"/lib --add-modules javafx.controls GUI.java
